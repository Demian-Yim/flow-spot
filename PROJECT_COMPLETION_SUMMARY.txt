PROJECT OVERVIEW
================================================================================

FLOW SPOT is a production-ready corporate training spot game platform built with:
- Next.js 14 (App Router)
- TypeScript (strict mode)
- Tailwind CSS with custom FLOW SPOT color palette
- Firebase Authentication & Firestore
- Google Gemini AI API
- Framer Motion animations
- Web Audio API for sound effects
- QR code generation

     - Session status and progress

  - Field-level permissions
  - Activity-specific access controls

FIRESTORE DOCUMENT SCHEMAS
================================================================================

projects:
{
  id: string,
  name: string,
  instructorId: string,
  sessionCode: string,
  createdAt: timestamp,
  status: "draft" | "active" | "completed",
  spotSequence: ["manner", "stretch", "greeting", "ground-rules", "mini-games"],
  settings: {
    enableSound: boolean,
    enableConfetti: boolean,
    enableAvatars: boolean,
    timePerSpot: number
  }
}

sessions:
{
  id: string,
  projectId: string,
  sessionCode: string,
  instructorId: string,
  createdAt: timestamp,
  startedAt?: timestamp,
  currentSpotIndex: number,
  participantCount: number,
  status: "waiting" | "active" | "completed"
}

participants:
{
  id: string,
  projectId: string,
  sessionCode: string,
  name: string,
  organization: string,
  title: string,
  avatar?: string,
  joinedAt: timestamp,
  responses: [
    {
      spotId: string,
      spotType: string,
      response: object,
      respondedAt: timestamp
    }
  ],
  status: "active" | "inactive"
}
