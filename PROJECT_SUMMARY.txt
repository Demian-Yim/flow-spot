================================================================================
                              PROJECT OVERVIEW
================================================================================

FLOW SPOT is a complete, production-ready Next.js application for corporate 
training game sessions. It enables instructors to run team-building exercises 
using interactive spot activities and mini-games.

Tech Stack:
- Frontend: Next.js 14+ (App Router)
- Styling: Tailwind CSS
- Animations: Framer Motion
- Backend: Firebase (Auth + Firestore)
- AI: Google Gemini API
- Deployment: Vercel
